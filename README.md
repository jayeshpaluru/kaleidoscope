# Color Theory Kaleidoscope

This project simulates color theory through a dynamic kaleidoscope effect. Users can manipulate the colors and visual effects using sliders and control panels.

## Project Structure

- **index.html**: The main HTML document that sets up the structure of the web page.
- **css/style.css**: Contains styles for the web page, including layout and visual effects.
- **js/main.js**: The entry point for JavaScript functionality, initializing the kaleidoscope effect and setting up event listeners.
- **js/kaleidoscope.js**: Logic for rendering the kaleidoscope effect based on RGB values from the sliders.
- **js/controls.js**: Manages control elements, including sliders for red, blue, and green, and squares for controlling visual effects.

## Features

- **Color Sliders**: Adjust the red, green, and blue components to change the kaleidoscope colors.
- **Tripiness Control**: Use the four squares to modify the visual effects and intensity of the kaleidoscope.

## How to Run the Project

1. Clone the repository or download the project files.
2. Open `index.html` in a web browser.
3. Use the sliders to adjust colors and the control panel to modify visual effects.

Enjoy experimenting with color theory and creating mesmerizing kaleidoscope visuals!
